Passos para Instalação do APK Condor Center

Passo 1: Preparação do Ambiente

Verifique se seu ambiente de desenvolvimento está configurado corretamente:

./prepare-mobile-environment.sh

Este script verificará se você tem todas as dependências necessárias instaladas e se os recursos gráficos estão disponíveis.

Passo 2: Construção do Aplicativo

Execute o script de build para compilar o aplicativo web e preparar os projetos nativos:

./capacitor-build.sh

Este processo irá:

  • Compilar a aplicação web
  • Sincronizar com o Capacitor
  • Criar ou atualizar os projetos Android e iOS
  • Gerar os recursos visuais necessários

Passo 3: Compilação do APK

Abra o projeto no Android Studio para compilar o APK:

npx cap open android

No Android Studio:

  1. Aguarde a sincronização do Gradle
  2. Selecione Build > Build Bundle(s) / APK(s) > Build APK(s)
  3. O APK será gerado em android/app/build/outputs/apk/debug/app-debug.apk

Alternativamente, compile via linha de comando:

cd android && ./gradlew assembleDebug

Passo 4: Instalação em Dispositivos

Use o script de instalação para instalar o APK em dispositivos conectados:

./install-mobile-apps.sh

Este script irá:

  • Verificar dispositivos Android conectados via USB
  • Instalar o APK em todos os dispositivos
  • Confirmar a instalação bem-sucedida

Alternativamente, transfira o APK manualmente para o dispositivo e instale-o.

Observações Importantes

  • Certifique-se de ter a depuração USB ativada no dispositivo Android
  • Para instalar em dispositivos iOS, você precisará de um Mac com Xcode
  • Para distribuição em produção, siga as instruções em MOBILE_APP_GUIDE.md
  • Consulte README-CONDOR-MOBILE.md para documentação completa
